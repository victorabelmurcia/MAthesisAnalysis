Folder Description

The directory is the main data container. There are followig files here:
  • Datasets:
    • DataLab202.csv - preprocessed file with data on indviduals (not places). It is prepared to be used and its description can be found in the Codebook ("Codebook.html")
    • raw211.csv - raw data (only anonymized) taken directly from the Google.docs survey.
    • eduprog2.csv - dataset used to classiffy the respondents' education types by hand.
    • Names.csv - dataset used to rename variable names from generic names in the google spreadsheet to more something more convenient.
    • occupation2.csv - dataset used to classify the respondents' current professions by hand.
    • trainfav2.csv - dataset used to classify the respondent's favourite sport activity by hand.
    • Places211.csv - raw data on places, that is which respondents visit which places.
  • Scripts and functions:
    • DataTransHelperFuncs.R - a helper function that imitates SPSS-like recoding of multiple variables of the same structure at once.
    • RelabellingAndBasicDataTrans.R - the script that generates DataLab202.csv from raw211.csv.
    • getColnamesDF.R - simple scripts that rename variable names.
